## Pourquoi construire un moteur clé-valeur ?

Comprendre le fonctionnement interne d'un moteur de stockage est fondamental pour tout architecte système. Redis, RocksDB, etcd — tous partagent des principes communs que nous allons implémenter de zéro.

---

## Architecture d'un KV Store durable

```
Écriture :
Client → [WAL (append-only)] → [Mémtable (HashMap)] → [Réponse OK]
                                        │
                                 (flush périodique)
                                        │
                                        ▼
                                 [SSTable sur disque]

Lecture :
Client → [Mémtable] → trouvé ? Réponse
               └── non trouvé → [SSTable] → Réponse
```

Le WAL (Write-Ahead Log) garantit la durabilité : chaque écriture est d'abord enregistrée dans un fichier séquentiel append-only avant d'être appliquée en mémoire. En cas de crash, on peut rejouer le WAL pour reconstruire l'état.

## Implémentation complète

```rust
// src/kv_store.rs
use std::collections::HashMap;
use std::io::{BufWriter, Write};
use std::path::{Path, PathBuf};
use std::fs::{File, OpenOptions};
use tokio::sync::RwLock;
use serde::{Deserialize, Serialize};
use std::sync::Arc;

#[derive(Debug, Serialize, Deserialize, Clone)]
pub enum EntréeWAL {
    Set { clé: String, valeur: Vec<u8> },
    Delete { clé: String },
    Checkpoint { horodatage: i64 },
}

#[derive(Debug, thiserror::Error)]
pub enum ErreurKV {
    #[error("Erreur I/O : {0}")]
    IO(#[from] std::io::Error),
    #[error("Sérialisation : {0}")]
    Sérialisation(#[from] bincode::Error),
    #[error("Clé introuvable : {0}")]
    Introuvable(String),
}

pub struct KVStore {
    mémtable: Arc<RwLock<HashMap<String, Option<Vec<u8>>>>>,
    wal: Arc<tokio::sync::Mutex<WalWriter>>,
    chemin: PathBuf,
}

struct WalWriter {
    fichier: BufWriter<File>,
    taille: u64,
    max_taille: u64,
}

impl WalWriter {
    fn nouveau(chemin: &Path, max_taille: u64) -> Result<Self, ErreurKV> {
        let fichier = OpenOptions::new()
            .create(true)
            .append(true)
            .open(chemin.join("wal.bin"))?;
        
        let taille = fichier.metadata()?.len();
        
        Ok(WalWriter {
            fichier: BufWriter::new(fichier),
            taille,
            max_taille,
        })
    }
    
    fn écrire(&mut self, entrée: &EntréeWAL) -> Result<(), ErreurKV> {
        let octets = bincode::serialize(entrée)?;
        let longueur = octets.len() as u32;
        
        // Format : [4 octets longueur][données]
        self.fichier.write_all(&longueur.to_le_bytes())?;
        self.fichier.write_all(&octets)?;
        self.fichier.flush()?; // fsync pour durabilité
        
        self.taille += 4 + octets.len() as u64;
        Ok(())
    }
    
    fn doit_compacter(&self) -> bool {
        self.taille >= self.max_taille
    }
}

impl KVStore {
    pub fn nouveau(chemin: impl AsRef<Path>) -> Result<Self, ErreurKV> {
        let chemin = chemin.as_ref().to_path_buf();
        std::fs::create_dir_all(&chemin)?;
        
        let mut mémtable = HashMap::new();
        
        // Rejouer le WAL pour reconstruire l'état
        let chemin_wal = chemin.join("wal.bin");
        if chemin_wal.exists() {
            rejouer_wal(&chemin_wal, &mut mémtable)?;
            tracing::info!("{} entrées chargées depuis le WAL", mémtable.len());
        }
        
        let wal = WalWriter::nouveau(&chemin, 64 * 1024 * 1024)?; // 64 Mo max
        
        Ok(KVStore {
            mémtable: Arc::new(RwLock::new(mémtable)),
            wal: Arc::new(tokio::sync::Mutex::new(wal)),
            chemin,
        })
    }
    
    pub async fn set(&self, clé: &str, valeur: Vec<u8>) -> Result<(), ErreurKV> {
        // 1. Écrire dans le WAL (durabilité)
        {
            let mut wal = self.wal.lock().await;
            wal.écrire(&EntréeWAL::Set {
                clé: clé.to_string(),
                valeur: valeur.clone(),
            })?;
            
            if wal.doit_compacter() {
                // Déclencher la compaction en arrière-plan
                drop(wal);
                self.compacter().await?;
            }
        }
        
        // 2. Mettre à jour la mémtable
        let mut table = self.mémtable.write().await;
        table.insert(clé.to_string(), Some(valeur));
        
        Ok(())
    }
    
    pub async fn get(&self, clé: &str) -> Result<Option<Vec<u8>>, ErreurKV> {
        let table = self.mémtable.read().await;
        Ok(table.get(clé).and_then(|v| v.clone()))
    }
    
    pub async fn delete(&self, clé: &str) -> Result<(), ErreurKV> {
        // Tombstone : marquer comme supprimé (pas retirer immédiatement)
        {
            let mut wal = self.wal.lock().await;
            wal.écrire(&EntréeWAL::Delete { clé: clé.to_string() })?;
        }
        
        let mut table = self.mémtable.write().await;
        table.insert(clé.to_string(), None); // None = tombstone
        
        Ok(())
    }
    
    async fn compacter(&self) -> Result<(), ErreurKV> {
        tracing::info!("Démarrage de la compaction WAL");
        
        // Snapshot de la mémtable actuelle
        let snapshot: Vec<(String, Vec<u8>)> = {
            let table = self.mémtable.read().await;
            table.iter()
                .filter_map(|(k, v)| v.as_ref().map(|val| (k.clone(), val.clone())))
                .collect()
        };
        
        // Écrire le snapshot sur disque
        let chemin_snapshot = self.chemin.join("snapshot.bin");
        let données = bincode::serialize(&snapshot)?;
        std::fs::write(&chemin_snapshot, &données)?;
        
        // Réinitialiser le WAL
        {
            let mut wal = self.wal.lock().await;
            let nouveau_wal = WalWriter::nouveau(&self.chemin, 64 * 1024 * 1024)?;
            // Écrire un checkpoint
            // *wal = nouveau_wal; // Remplacer l'ancien WAL
        }
        
        tracing::info!("Compaction terminée : {} entrées", snapshot.len());
        Ok(())
    }
}

fn rejouer_wal(
    chemin: &Path,
    mémtable: &mut HashMap<String, Option<Vec<u8>>>,
) -> Result<(), ErreurKV> {
    use std::io::Read;
    
    let mut fichier = File::open(chemin)?;
    let mut tampon_longueur = [0u8; 4];
    
    loop {
        match fichier.read_exact(&mut tampon_longueur) {
            Ok(()) => {}
            Err(e) if e.kind() == std::io::ErrorKind::UnexpectedEof => break,
            Err(e) => return Err(ErreurKV::IO(e)),
        }
        
        let longueur = u32::from_le_bytes(tampon_longueur) as usize;
        let mut données = vec![0u8; longueur];
        fichier.read_exact(&mut données)?;
        
        let entrée: EntréeWAL = bincode::deserialize(&données)?;
        
        match entrée {
            EntréeWAL::Set { clé, valeur } => {
                mémtable.insert(clé, Some(valeur));
            }
            EntréeWAL::Delete { clé } => {
                mémtable.insert(clé, None);
            }
            EntréeWAL::Checkpoint { .. } => {
                // Le checkpoint indique qu'un snapshot a été fait avant ce point
                // Tout ce qui précède est sûr d'être ignoré
            }
        }
    }
    
    Ok(())
}
```

## Sharding — Partitionner les données

```rust
use std::collections::hash_map::DefaultHasher;
use std::hash::{Hash, Hasher};

pub struct KVShardé {
    shards: Vec<KVStore>,
    nb_shards: usize,
}

impl KVShardé {
    pub async fn nouveau(répertoire: &Path, nb_shards: usize) -> Result<Self, ErreurKV> {
        let mut shards = Vec::with_capacity(nb_shards);
        
        for i in 0..nb_shards {
            let chemin_shard = répertoire.join(format!("shard-{:03}", i));
            shards.push(KVStore::nouveau(chemin_shard)?);
        }
        
        Ok(KVShardé { shards, nb_shards })
    }
    
    fn shard_pour(&self, clé: &str) -> usize {
        // Hashing cohérent simple
        let mut hasher = DefaultHasher::new();
        clé.hash(&mut hasher);
        (hasher.finish() as usize) % self.nb_shards
    }
    
    pub async fn set(&self, clé: &str, valeur: Vec<u8>) -> Result<(), ErreurKV> {
        let shard_idx = self.shard_pour(clé);
        self.shards[shard_idx].set(clé, valeur).await
    }
    
    pub async fn get(&self, clé: &str) -> Result<Option<Vec<u8>>, ErreurKV> {
        let shard_idx = self.shard_pour(clé);
        self.shards[shard_idx].get(clé).await
    }
}
```
