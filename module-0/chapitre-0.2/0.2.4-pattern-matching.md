## Le pattern matching va bien au-delà du switch/case

Si vous venez d'un autre langage, vous êtes probablement habitué au `switch/case` — un outil utile mais limité. Le `match` de Rust est une bête d'une autre catégorie : il est **exhaustif**, **déstructurant**, et capable de traiter des structures imbriquées d'une profondeur arbitraire.

---

## Déstructuration — Extraire les données directement

```rust
struct Point { x: f64, y: f64 }
struct Rect { coin_hg: Point, coin_bd: Point }

fn analyser_rect(r: &Rect) {
    // Déstructuration de struct imbriquée
    let Rect {
        coin_hg: Point { x: x1, y: y1 },
        coin_bd: Point { x: x2, y: y2 },
    } = r;
    
    println!("Largeur: {}, Hauteur: {}", (x2 - x1).abs(), (y2 - y1).abs());
}

fn main() {
    // Déstructuration dans let
    let (a, b, c) = (1, 2, 3);
    
    // Déstructuration dans les arguments de fonction
    let points = vec![Point { x: 1.0, y: 2.0 }, Point { x: 3.0, y: 4.0 }];
    let somme_x: f64 = points.iter().map(|Point { x, .. }| x).sum();
    
    // Déstructuration avec renommage
    let Point { x: abscisse, y: ordonnée } = Point { x: 5.0, y: 10.0 };
    println!("abscisse={}, ordonnée={}", abscisse, ordonnée);
}
```

## Patterns sur les enums — La puissance réelle

```rust
#[derive(Debug)]
enum Commande {
    DémarrerServeur { hôte: String, port: u16 },
    ArrêterServeur,
    Recharger { fichier_config: String },
    Ping { id: u32 },
    Pong { id: u32 },
}

fn exécuter(cmd: Commande) {
    match cmd {
        // Déstructuration complète
        Commande::DémarrerServeur { hôte, port } => {
            println!("Démarrage sur {}:{}", hôte, port);
        }
        
        // Capture avec guard
        Commande::Ping { id } if id == 0 => {
            println!("Ping système reçu (id=0)");
        }
        Commande::Ping { id } => {
            println!("Ping {} — envoi Pong", id);
        }
        
        // Pattern OR
        Commande::ArrêterServeur | Commande::Recharger { .. } => {
            println!("Opération de maintenance");
        }
        
        // Ignorer les champs non nécessaires
        Commande::Pong { .. } => {
            println!("Pong ignoré");
        }
    }
}
```

## if let et while let — Pour les cas simples

```rust
fn traiter_optionnel(opt: Option<i32>) {
    // if let — quand vous n'avez qu'un seul variant qui vous intéresse
    if let Some(val) = opt {
        println!("Valeur : {}", val);
    } else {
        println!("Rien");
    }
    
    // Équivalent avec match — plus verbeux
    match opt {
        Some(val) => println!("Valeur : {}", val),
        None => println!("Rien"),
    }
}

fn traiter_file(mut file: std::collections::VecDeque<Message>) {
    // while let — tant que le pattern matche
    while let Some(message) = file.pop_front() {
        traiter(message);
    }
}
```

## Binding avec @ — Capturer et tester simultanément

```rust
fn classifier_âge(âge: u32) -> &'static str {
    match âge {
        // @ capture la valeur ET vérifie la plage
        n @ 0..=12 => {
            println!("Enfant de {} ans", n);
            "enfant"
        }
        n @ 13..=17 => {
            println!("Adolescent de {} ans", n);
            "adolescent"
        }
        n @ 18..=64 => {
            println!("Adulte de {} ans", n);
            "adulte"
        }
        n => {
            println!("Senior de {} ans", n);
            "senior"
        }
    }
}
```

## Le pattern "typestate" — Compiler les invariants

Ce pattern encode les transitions d'état valides directement dans le système de types. Les transitions invalides deviennent des **erreurs de compilation**.

```rust
// Types d'état — structs vides (zero-cost)
struct NonConnecté;
struct Connecté;
struct Authentifié;

// La connexion est générique sur son état
struct Connexion<État> {
    adresse: String,
    _état: std::marker::PhantomData<État>,
}

// Méthodes disponibles SEULEMENT à l'état NonConnecté
impl Connexion<NonConnecté> {
    fn nouvelle(adresse: &str) -> Self {
        Connexion {
            adresse: adresse.to_string(),
            _état: std::marker::PhantomData,
        }
    }
    
    fn connecter(self) -> Result<Connexion<Connecté>, String> {
        println!("Connexion à {}", self.adresse);
        // Simulation connexion réseau...
        Ok(Connexion {
            adresse: self.adresse,
            _état: std::marker::PhantomData,
        })
    }
}

// Méthodes disponibles SEULEMENT à l'état Connecté
impl Connexion<Connecté> {
    fn authentifier(self, token: &str) -> Result<Connexion<Authentifié>, String> {
        if token == "secret" {
            Ok(Connexion {
                adresse: self.adresse,
                _état: std::marker::PhantomData,
            })
        } else {
            Err("Token invalide".to_string())
        }
    }
    
    fn déconnecter(self) -> Connexion<NonConnecté> {
        Connexion {
            adresse: self.adresse,
            _état: std::marker::PhantomData,
        }
    }
}

// Méthodes disponibles SEULEMENT après authentification
impl Connexion<Authentifié> {
    fn envoyer_données(&self, données: &str) {
        println!("Envoi sécurisé vers {} : {}", self.adresse, données);
    }
    
    fn déconnecter(self) -> Connexion<NonConnecté> {
        Connexion {
            adresse: self.adresse,
            _état: std::marker::PhantomData,
        }
    }
}

fn main() {
    let conn = Connexion::<NonConnecté>::nouvelle("localhost:8080");
    
    // conn.envoyer_données("test"); // ← ERREUR DE COMPILATION !
    // Méthode inexistante sur Connexion<NonConnecté>
    
    let conn = conn.connecter().unwrap();
    
    // conn.envoyer_données("test"); // ← ERREUR DE COMPILATION !
    // Méthode inexistante sur Connexion<Connecté>
    
    let conn = conn.authentifier("secret").unwrap();
    
    conn.envoyer_données("données sensibles"); // ✅ Maintenant valide
}
```

Ce pattern est utilisé dans des bibliothèques de production comme `tokio::net::TcpStream` et des clients de bases de données.

## Matches! — La macro de test rapide

```rust
fn est_erreur_client(code: u16) -> bool {
    matches!(code, 400..=499)
}

fn est_weekend(jour: &str) -> bool {
    matches!(jour, "samedi" | "dimanche")
}

// Dans des filtres
let codes_erreur: Vec<u16> = vec![200, 404, 500, 403, 201];
let erreurs_client: Vec<u16> = codes_erreur
    .into_iter()
    .filter(|&c| matches!(c, 400..=499))
    .collect();
// [404, 403]
```

## 🛠️ En production — Router de messages distribués

```rust
#[derive(Debug)]
enum MessageDistribué {
    HeartBeat { nœud_id: u32, horodatage: u64 },
    Données { partition: u16, clé: Vec<u8>, valeur: Vec<u8> },
    Contrôle { commande: String, paramètres: Vec<String> },
    Erreur { code: u32, message: String, fatal: bool },
}

fn router_message(msg: MessageDistribué) -> Option<String> {
    match msg {
        // Filtrer les heartbeats silencieux
        MessageDistribué::HeartBeat { .. } => None,
        
        // Router les données selon la partition
        MessageDistribué::Données { partition, ref clé, .. } if partition < 16 => {
            Some(format!("nœud-{}", partition % 4))
        }
        MessageDistribué::Données { partition, .. } => {
            Some(format!("nœud-overflow-{}", partition))
        }
        
        // Arrêt immédiat sur erreur fatale
        MessageDistribué::Erreur { fatal: true, message, .. } => {
            eprintln!("ERREUR FATALE : {}", message);
            std::process::exit(1);
        }
        
        // Autres erreurs — log et continuer
        MessageDistribué::Erreur { code, message, .. } => {
            eprintln!("Erreur {}: {}", code, message);
            None
        }
        
        MessageDistribué::Contrôle { commande, .. } => {
            Some(format!("contrôleur-{}", commande))
        }
    }
}
```

## Résumé

| Pattern | Syntaxe | Usage |
|---|---|---|
| Déstructuration | `let Point { x, y } = p` | Extraire les champs |
| Guard | `Some(n) if n > 0` | Condition supplémentaire |
| Pattern OR | `A \| B => ...` | Plusieurs variants ensemble |
| Binding | `n @ 1..=10` | Capturer + tester |
| `if let` | `if let Some(v) = opt` | Un seul variant |
| `while let` | `while let Some(v) = iter.next()` | Boucle sur pattern |
| Typestate | `Connexion<Authentifié>` | États valides compilés |
| `matches!` | `matches!(v, A \| B)` | Test boolean rapide |
