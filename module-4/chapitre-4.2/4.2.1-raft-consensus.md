## Pourquoi Raft ?

Paxos, le premier algorithme de consensus, est notoire pour sa complexité. Raft a été conçu explicitement pour être **compréhensible** — et c'est sa grande force.

Raft décompose le problème du consensus en trois sous-problèmes indépendants :
1. **Leader election** : élire un leader parmi les nœuds
2. **Log replication** : le leader reçoit toutes les écritures et les réplique
3. **Safety** : garantir qu'un log accepté ne sera jamais perdu

```
Cluster Raft (5 nœuds) :
┌─────────┐    ┌─────────┐    ┌─────────┐
│ Follower│    │ LEADER  │    │ Follower│
│ Nœud 1  │◄───│ Nœud 2  │───►│ Nœud 3  │
└─────────┘    └────┬────┘    └─────────┘
                    │
     ┌──────────────┴──────────────┐
     ▼                             ▼
┌─────────┐                  ┌─────────┐
│ Follower│                  │ Follower│
│ Nœud 4  │                  │ Nœud 5  │
└─────────┘                  └─────────┘

Toutes les écritures passent par le Leader.
Le Leader réplique sur les Followers.
Si 3/5 nœuds confirment → commit.
```

## Les états d'un nœud Raft

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum ÉtatRaft {
    Follower,
    Candidate,
    Leader,
}

/// Transition d'état :
///
/// Démarrage → Follower
/// Follower → Candidate (timeout sans heartbeat du Leader)
/// Candidate → Leader (reçoit votes de la majorité)
/// Candidate → Follower (découvre un Leader valide)
/// Leader → Follower (découvre un terme plus récent)
```

## Types de messages Raft

```rust
use serde::{Deserialize, Serialize};

pub type NœudId = u32;
pub type Terme = u64;
pub type IndexLog = u64;

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct EntréeLog {
    pub terme: Terme,
    pub index: IndexLog,
    pub commande: Vec<u8>,
}

/// RequestVote : demande de vote lors d'une élection
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RequêteVote {
    pub terme: Terme,
    pub candidat_id: NœudId,
    pub dernier_index_log: IndexLog,
    pub dernier_terme_log: Terme,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RéponseVote {
    pub terme: Terme,
    pub vote_accordé: bool,
}

/// AppendEntries : heartbeat + réplication du log
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RequêteAppendEntries {
    pub terme: Terme,
    pub leader_id: NœudId,
    pub index_précédent: IndexLog,
    pub terme_précédent: Terme,
    pub entrées: Vec<EntréeLog>,
    pub commit_index: IndexLog,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct RéponseAppendEntries {
    pub terme: Terme,
    pub succès: bool,
    pub prochain_index: IndexLog, // Pour optimiser les retry
}
```

## État d'un nœud Raft

```rust
use std::collections::HashMap;
use tokio::sync::{mpsc, watch};

pub struct NœudRaft {
    // Identité
    pub id: NœudId,
    pub pairs: Vec<NœudId>,
    
    // État persistant (à sauvegarder sur disque)
    pub terme_actuel: Terme,
    pub voté_pour: Option<NœudId>,
    pub log: Vec<EntréeLog>,
    
    // État volatile
    pub état: ÉtatRaft,
    pub commit_index: IndexLog,
    pub dernier_appliqué: IndexLog,
    
    // État du Leader (valide seulement si état == Leader)
    pub prochain_index: HashMap<NœudId, IndexLog>,
    pub match_index: HashMap<NœudId, IndexLog>,
    
    // Communication
    pub tx_sortant: mpsc::Sender<(NœudId, MessageRaft)>,
    pub tx_état: watch::Sender<ÉtatRaft>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub enum MessageRaft {
    RequêteVote(RequêteVote),
    RéponseVote(RéponseVote),
    AppendEntries(RequêteAppendEntries),
    RéponseAppendEntries(RéponseAppendEntries),
}
```

## Le cœur de l'algorithme : traitement des messages

```rust
impl NœudRaft {
    /// Traiter un message entrant
    pub async fn traiter_message(
        &mut self,
        émetteur: NœudId,
        message: MessageRaft,
    ) {
        match message {
            MessageRaft::RequêteVote(req) => {
                self.traiter_requête_vote(émetteur, req).await;
            }
            MessageRaft::RéponseVote(rep) => {
                self.traiter_réponse_vote(émetteur, rep).await;
            }
            MessageRaft::AppendEntries(req) => {
                self.traiter_append_entries(émetteur, req).await;
            }
            MessageRaft::RéponseAppendEntries(rep) => {
                self.traiter_réponse_append(émetteur, rep).await;
            }
        }
    }
    
    async fn traiter_requête_vote(&mut self, candidat: NœudId, req: RequêteVote) {
        // Règle 1 : si le terme est plus récent, devenir Follower
        if req.terme > self.terme_actuel {
            self.devenir_follower(req.terme).await;
        }
        
        let vote_accordé = self.peut_voter_pour(candidat, &req);
        
        if vote_accordé {
            self.voté_pour = Some(candidat);
            tracing::debug!("Vote accordé à nœud {}", candidat);
        }
        
        let réponse = RéponseVote {
            terme: self.terme_actuel,
            vote_accordé,
        };
        
        let _ = self.tx_sortant.send((candidat, MessageRaft::RéponseVote(réponse))).await;
    }
    
    fn peut_voter_pour(&self, candidat: NœudId, req: &RequêteVote) -> bool {
        // Conditions pour accorder le vote :
        
        // 1. Le terme doit être actuel
        if req.terme < self.terme_actuel {
            return false;
        }
        
        // 2. N'avoir pas encore voté (ou avoir voté pour ce candidat)
        let peut_voter = self.voté_pour.is_none()
            || self.voté_pour == Some(candidat);
        
        if !peut_voter { return false; }
        
        // 3. Le log du candidat doit être au moins aussi à jour
        let mon_dernier_terme = self.log.last().map_or(0, |e| e.terme);
        let mon_dernier_index = self.log.len() as IndexLog;
        
        if req.dernier_terme_log != mon_dernier_terme {
            return req.dernier_terme_log > mon_dernier_terme;
        }
        
        req.dernier_index_log >= mon_dernier_index
    }
    
    async fn traiter_append_entries(&mut self, leader: NœudId, req: RequêteAppendEntries) {
        // Rejeter si terme obsolète
        if req.terme < self.terme_actuel {
            let réponse = RéponseAppendEntries {
                terme: self.terme_actuel,
                succès: false,
                prochain_index: 0,
            };
            let _ = self.tx_sortant.send((leader, MessageRaft::RéponseAppendEntries(réponse))).await;
            return;
        }
        
        // Réinitialiser le timer d'élection (on a des nouvelles du Leader)
        self.devenir_follower(req.terme).await;
        
        // Vérifier la cohérence du log
        let cohérent = req.index_précédent == 0 || {
            self.log.get((req.index_précédent - 1) as usize)
                .map_or(false, |e| e.terme == req.terme_précédent)
        };
        
        if !cohérent {
            let réponse = RéponseAppendEntries {
                terme: self.terme_actuel,
                succès: false,
                prochain_index: self.log.len() as IndexLog,
            };
            let _ = self.tx_sortant.send((leader, MessageRaft::RéponseAppendEntries(réponse))).await;
            return;
        }
        
        // Ajouter les nouvelles entrées
        let début = req.index_précédent as usize;
        self.log.truncate(début); // Supprimer les entrées conflictuelles
        self.log.extend(req.entrées.into_iter());
        
        // Mettre à jour le commit index
        if req.commit_index > self.commit_index {
            self.commit_index = req.commit_index.min(self.log.len() as IndexLog);
            self.appliquer_commandes_committées().await;
        }
        
        let réponse = RéponseAppendEntries {
            terme: self.terme_actuel,
            succès: true,
            prochain_index: self.log.len() as IndexLog + 1,
        };
        let _ = self.tx_sortant.send((leader, MessageRaft::RéponseAppendEntries(réponse))).await;
    }
    
    async fn appliquer_commandes_committées(&mut self) {
        while self.dernier_appliqué < self.commit_index {
            self.dernier_appliqué += 1;
            let entrée = &self.log[(self.dernier_appliqué - 1) as usize];
            
            // Appliquer la commande à la machine d'état
            tracing::debug!(
                index = self.dernier_appliqué,
                "Application de la commande committée"
            );
            // état_machine.appliquer(&entrée.commande).await;
        }
    }
    
    async fn devenir_follower(&mut self, terme: Terme) {
        self.terme_actuel = terme;
        self.voté_pour = None;
        self.état = ÉtatRaft::Follower;
        let _ = self.tx_état.send(ÉtatRaft::Follower);
    }
}
```

## 🔥 Failure First — Tester la tolérance aux pannes

```rust
#[tokio::test]
async fn cluster_survit_à_la_perte_du_leader() {
    let mut cluster = ClusterTest::démarrer(5).await;
    
    // Attendre l'élection initiale
    let leader = cluster.attendre_leader(Duration::from_secs(5)).await.unwrap();
    println!("Leader élu : nœud {}", leader);
    
    // Écrire une valeur
    cluster.écrire(leader, b"clé", b"valeur-1").await.unwrap();
    
    // Tuer le leader
    println!("Simulation de panne du leader...");
    cluster.tuer_nœud(leader).await;
    
    // Attendre une nouvelle élection (devrait prendre < 1 seconde)
    let nouveau_leader = cluster
        .attendre_leader(Duration::from_secs(3))
        .await
        .expect("Nouveau leader doit être élu");
    
    assert_ne!(nouveau_leader, leader, "Le nouveau leader doit être différent");
    println!("Nouveau leader : nœud {}", nouveau_leader);
    
    // Vérifier que la valeur est toujours disponible
    let valeur = cluster.lire(nouveau_leader, b"clé").await.unwrap();
    assert_eq!(valeur, b"valeur-1");
    
    // Écrire une nouvelle valeur
    cluster.écrire(nouveau_leader, b"clé", b"valeur-2").await.unwrap();
    
    // Redémarrer l'ancien leader — il doit se synchroniser
    cluster.redémarrer_nœud(leader).await;
    tokio::time::sleep(Duration::from_secs(1)).await;
    
    // Vérifier la convergence
    let valeur_ancien = cluster.lire(leader, b"clé").await.unwrap();
    assert_eq!(valeur_ancien, b"valeur-2", "L'ancien leader doit se synchroniser");
}
```
