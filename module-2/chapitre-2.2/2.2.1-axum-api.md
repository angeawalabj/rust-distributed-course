## Pourquoi Axum ?

Axum est le framework HTTP de l'écosystème Tokio. Il est conçu pour être composable, type-safe et performant. Contrairement à Actix-Web (qui utilise son propre runtime), Axum est construit directement sur Tower (middleware ecosystem) et Hyper (HTTP).

```toml
[dependencies]
axum = "0.7"
tokio = { version = "1", features = ["full"] }
serde = { version = "1", features = ["derive"] }
serde_json = "1"
tower = "0.4"
tower-http = { version = "0.5", features = ["cors", "trace", "limit"] }
tracing = "0.1"
tracing-subscriber = "0.3"
uuid = { version = "1", features = ["v4"] }
validator = { version = "0.18", features = ["derive"] }
```

## Structure d'une application Axum

```rust
// src/main.rs
use axum::{
    routing::{get, post, put, delete},
    Router,
};
use std::sync::Arc;
use tower_http::{cors::CorsLayer, trace::TraceLayer};

mod erreurs;
mod handlers;
mod modèles;
mod état;

use état::ÉtatApp;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    tracing_subscriber::fmt()
        .with_env_filter("mon_api=debug,tower_http=debug")
        .init();
    
    let état = Arc::new(ÉtatApp::nouveau());
    
    let app = Router::new()
        // Routes publiques
        .route("/santé", get(handlers::santé::vérifier))
        .route("/auth/connexion", post(handlers::auth::connexion))
        .route("/auth/inscription", post(handlers::auth::inscription))
        // Routes protégées
        .route("/utilisateurs", get(handlers::utilisateurs::lister))
        .route("/utilisateurs/:id", get(handlers::utilisateurs::obtenir))
        .route("/utilisateurs/:id", put(handlers::utilisateurs::mettre_à_jour))
        .route("/utilisateurs/:id", delete(handlers::utilisateurs::supprimer))
        // Middlewares globaux
        .layer(TraceLayer::new_for_http())
        .layer(CorsLayer::permissive())
        .layer(axum::extract::DefaultBodyLimit::max(1024 * 1024)) // 1 Mo max
        .with_state(état);
    
    let addr = "0.0.0.0:8080";
    let listener = tokio::net::TcpListener::bind(addr).await?;
    tracing::info!("API démarrée sur http://{}", addr);
    
    axum::serve(listener, app).await?;
    Ok(())
}
```

## Handlers — Extraire et retourner des données

```rust
// src/handlers/utilisateurs.rs
use axum::{
    extract::{Path, Query, State},
    http::StatusCode,
    response::Json,
    Extension,
};
use serde::{Deserialize, Serialize};
use std::sync::Arc;
use uuid::Uuid;

use crate::{état::ÉtatApp, erreurs::ErreurAPI, modèles::Utilisateur};

// Paramètres de liste avec pagination
#[derive(Deserialize)]
pub struct ParamètresListe {
    #[serde(default = "défaut_page")]
    page: u32,
    #[serde(default = "défaut_par_page")]
    par_page: u32,
    recherche: Option<String>,
}

fn défaut_page() -> u32 { 1 }
fn défaut_par_page() -> u32 { 20 }

// GET /utilisateurs?page=2&par_page=10&recherche=alice
pub async fn lister(
    State(état): State<Arc<ÉtatApp>>,
    Query(params): Query<ParamètresListe>,
    Extension(utilisateur_connecté): Extension<UtilisateurConnecté>,
) -> Result<Json<RéponseListePaginée<Utilisateur>>, ErreurAPI> {
    
    tracing::info!(
        page = params.page,
        par_page = params.par_page,
        "Liste des utilisateurs demandée"
    );
    
    let résultat = état.store.lister(params.page, params.par_page).await?;
    
    Ok(Json(RéponseListePaginée {
        données: résultat.données,
        pagination: Pagination {
            page: params.page,
            par_page: params.par_page,
            total: résultat.total,
        },
    }))
}

// GET /utilisateurs/:id
pub async fn obtenir(
    State(état): State<Arc<ÉtatApp>>,
    Path(id): Path<Uuid>,
) -> Result<Json<Utilisateur>, ErreurAPI> {
    let utilisateur = état.store.obtenir_par_id(id).await?;
    Ok(Json(utilisateur))
}

// Corps d'une requête de mise à jour
#[derive(Deserialize, validator::Validate)]
pub struct RequêteMiseÀJour {
    #[validate(length(min = 2, max = 100))]
    nom: Option<String>,
    
    #[validate(email)]
    email: Option<String>,
}

// PUT /utilisateurs/:id
pub async fn mettre_à_jour(
    State(état): State<Arc<ÉtatApp>>,
    Path(id): Path<Uuid>,
    Json(corps): Json<RequêteMiseÀJour>,
) -> Result<Json<Utilisateur>, ErreurAPI> {
    use validator::Validate;
    corps.validate().map_err(ErreurAPI::ValidationÉchouée)?;
    
    let mis_à_jour = état.store.mettre_à_jour(id, corps.nom, corps.email).await?;
    Ok(Json(mis_à_jour))
}

// DELETE /utilisateurs/:id → 204 No Content
pub async fn supprimer(
    State(état): State<Arc<ÉtatApp>>,
    Path(id): Path<Uuid>,
) -> Result<StatusCode, ErreurAPI> {
    état.store.supprimer(id).await?;
    Ok(StatusCode::NO_CONTENT)
}

#[derive(Serialize)]
pub struct RéponseListePaginée<T> {
    pub données: Vec<T>,
    pub pagination: Pagination,
}

#[derive(Serialize)]
pub struct Pagination {
    pub page: u32,
    pub par_page: u32,
    pub total: u64,
}
```

## Gestion des erreurs Axum — IntoResponse

```rust
// src/erreurs.rs
use axum::{
    http::StatusCode,
    response::{IntoResponse, Json, Response},
};
use serde_json::json;
use thiserror::Error;

#[derive(Error, Debug)]
pub enum ErreurAPI {
    #[error("Ressource introuvable")]
    Introuvable,
    
    #[error("Non authentifié")]
    NonAuthentifié,
    
    #[error("Accès interdit")]
    Interdit,
    
    #[error("Données invalides")]
    ValidationÉchouée(#[from] validator::ValidationErrors),
    
    #[error("Conflit : {0}")]
    Conflit(String),
    
    #[error("Erreur interne")]
    Interne(#[from] anyhow::Error),
}

// Conversion automatique en réponse HTTP
impl IntoResponse for ErreurAPI {
    fn into_response(self) -> Response {
        let (statut, code, message) = match &self {
            ErreurAPI::Introuvable => (
                StatusCode::NOT_FOUND,
                "NOT_FOUND",
                "Ressource introuvable".to_string(),
            ),
            ErreurAPI::NonAuthentifié => (
                StatusCode::UNAUTHORIZED,
                "UNAUTHORIZED",
                "Authentification requise".to_string(),
            ),
            ErreurAPI::Interdit => (
                StatusCode::FORBIDDEN,
                "FORBIDDEN",
                "Accès interdit".to_string(),
            ),
            ErreurAPI::ValidationÉchouée(e) => (
                StatusCode::UNPROCESSABLE_ENTITY,
                "VALIDATION_ERROR",
                e.to_string(),
            ),
            ErreurAPI::Conflit(msg) => (
                StatusCode::CONFLICT,
                "CONFLICT",
                msg.clone(),
            ),
            ErreurAPI::Interne(e) => {
                // NE PAS exposer les détails de l'erreur interne
                tracing::error!("Erreur interne : {:?}", e);
                (
                    StatusCode::INTERNAL_SERVER_ERROR,
                    "INTERNAL_ERROR",
                    "Une erreur interne s'est produite".to_string(),
                )
            }
        };
        
        (statut, Json(json!({
            "erreur": {
                "code": code,
                "message": message,
            }
        }))).into_response()
    }
}
```

## Middleware d'authentification JWT

```rust
// src/middleware/auth.rs
use axum::{
    extract::{Request, State},
    http::header::AUTHORIZATION,
    middleware::Next,
    response::Response,
};
use std::sync::Arc;

use crate::{erreurs::ErreurAPI, état::ÉtatApp};

#[derive(Clone, Debug)]
pub struct UtilisateurConnecté {
    pub id: uuid::Uuid,
    pub email: String,
    pub rôle: String,
}

pub async fn middleware_auth(
    State(état): State<Arc<ÉtatApp>>,
    mut requête: Request,
    suivant: Next,
) -> Result<Response, ErreurAPI> {
    let token = extraire_token(&requête)?;
    let claims = état.jwt.vérifier(&token)?;
    
    // Injecter l'utilisateur dans les extensions de la requête
    requête.extensions_mut().insert(UtilisateurConnecté {
        id: claims.sub.parse().map_err(|_| ErreurAPI::NonAuthentifié)?,
        email: claims.email,
        rôle: claims.rôle,
    });
    
    Ok(suivant.run(requête).await)
}

fn extraire_token(requête: &Request) -> Result<String, ErreurAPI> {
    let header = requête
        .headers()
        .get(AUTHORIZATION)
        .ok_or(ErreurAPI::NonAuthentifié)?
        .to_str()
        .map_err(|_| ErreurAPI::NonAuthentifié)?;
    
    header
        .strip_prefix("Bearer ")
        .map(|s| s.to_string())
        .ok_or(ErreurAPI::NonAuthentifié)
}
```

## Routes avec authentification

```rust
use axum::middleware;

let app = Router::new()
    // Routes publiques (sans auth)
    .route("/santé", get(handlers::santé::vérifier))
    .route("/auth/connexion", post(handlers::auth::connexion))
    // Routes protégées (avec auth)
    .nest("/api/v1", 
        Router::new()
            .route("/utilisateurs", get(lister).post(créer))
            .route("/utilisateurs/:id", 
                get(obtenir).put(mettre_à_jour).delete(supprimer)
            )
            .layer(middleware::from_fn_with_state(
                état.clone(),
                middleware_auth,
            ))
    )
    .with_state(état);
```

## 🔥 Failure First — Injection d'erreurs réseau

```rust
// Middleware de chaos pour tester la résilience des clients
use rand::Rng;

async fn middleware_chaos(
    requête: Request,
    suivant: Next,
) -> Result<Response, StatusCode> {
    let mut rng = rand::thread_rng();
    
    // 5% du temps : retourner une erreur 500
    if rng.gen_bool(0.05) {
        tracing::warn!("🔥 Chaos : injection d'une erreur 500");
        return Err(StatusCode::INTERNAL_SERVER_ERROR);
    }
    
    // 10% du temps : délai artificiel (simule une lenteur)
    if rng.gen_bool(0.10) {
        let délai = rng.gen_range(500..2000);
        tracing::warn!("🔥 Chaos : délai artificiel de {}ms", délai);
        tokio::time::sleep(std::time::Duration::from_millis(délai)).await;
    }
    
    Ok(suivant.run(requête).await)
}
```

Activez ce middleware en développement pour tester que vos clients gèrent correctement les erreurs.

## Tests d'intégration pour l'API

```rust
// tests/api_tests.rs
use axum::http::StatusCode;
use axum_test::TestServer;

#[tokio::test]
async fn créer_utilisateur_valide() {
    let app = créer_app_test().await;
    let server = TestServer::new(app).unwrap();
    
    let réponse = server
        .post("/api/v1/utilisateurs")
        .json(&serde_json::json!({
            "nom": "Alice Martin",
            "email": "alice@test.com"
        }))
        .await;
    
    réponse.assert_status(StatusCode::CREATED);
    let corps: serde_json::Value = réponse.json();
    assert_eq!(corps["nom"], "Alice Martin");
}

#[tokio::test]
async fn accès_sans_token_retourne_401() {
    let app = créer_app_test().await;
    let server = TestServer::new(app).unwrap();
    
    let réponse = server.get("/api/v1/utilisateurs").await;
    réponse.assert_status(StatusCode::UNAUTHORIZED);
}
```
