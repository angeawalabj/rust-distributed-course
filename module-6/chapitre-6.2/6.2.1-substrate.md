## Substrate — Le framework blockchain en Rust

Substrate est le framework développé par Parity Technologies pour construire des blockchains personnalisées. Polkadot, Kusama, et des centaines d'autres chaînes sont construites avec Substrate.

```
Architecture Substrate :
┌─────────────────────────────────────────┐

│  │         Runtime WASM            │    │
│  │  ┌────────┐  ┌──────────────┐   │    │
│  │  │ Pallets│  │Smart Contracts│  │    │
```

## Créer un Pallet personnalisé

Les pallets sont les modules qui composent le runtime Substrate. Voici un pallet de registre de ressources :

```rust
// pallets/registre-ressources/src/lib.rs
#![cfg_attr(not(feature = "std"), no_std)]

pub use pallet::*;

#[frame_support::pallet]
pub mod pallet {
    use frame_support::{
        pallet_prelude::*,
        traits::Currency,
    };
    use frame_system::pallet_prelude::*;
    use sp_std::vec::Vec;

    type BalanceOf<T> = <<T as Config>::Currency as Currency<
        <T as frame_system::Config>::AccountId
    >>::Balance;

    #[pallet::config]
    pub trait Config: frame_system::Config {
        type RuntimeEvent: From<Event<Self>> + IsType<<Self as frame_system::Config>::RuntimeEvent>;
        type Currency: Currency<Self::AccountId>;
        #[pallet::constant]
        type FraisEnregistrement: Get<BalanceOf<Self>>;
    }

    // Stockage des ressources
    #[pallet::storage]
    #[pallet::getter(fn ressources)]
    pub type Ressources<T: Config> = StorageMap<
        _,
        Blake2_128Concat,
        Vec<u8>,                    // identifiant de la ressource
        (T::AccountId, T::BlockNumber), // propriétaire, bloc d'enregistrement
        OptionQuery,
    >;

    #[pallet::event]
    #[pallet::generate_deposit(pub(super) fn deposit_event)]
    pub enum Event<T: Config> {
        RessourceEnregistrée {
            identifiant: Vec<u8>,
            propriétaire: T::AccountId,
        },
        RessourceTransférée {
            identifiant: Vec<u8>,
            ancien_propriétaire: T::AccountId,
            nouveau_propriétaire: T::AccountId,
        },
    }

    #[pallet::error]
    pub enum Error<T> {
        RessourceDéjàEnregistrée,
        RessourceIntrouvable,
        NonAutoriséÀTransférer,
        IdentifiantTropLong,
    }

    #[pallet::call]
    impl<T: Config> Pallet<T> {
        /// Enregistrer une nouvelle ressource
        #[pallet::call_index(0)]
        #[pallet::weight(10_000)]
        pub fn enregistrer(
            origin: OriginFor<T>,
            identifiant: Vec<u8>,
        ) -> DispatchResult {
            let appelant = ensure_signed(origin)?;

            ensure!(identifiant.len() <= 64, Error::<T>::IdentifiantTropLong);
            ensure!(
                !Ressources::<T>::contains_key(&identifiant),
                Error::<T>::RessourceDéjàEnregistrée
            );

            // Prélever les frais
            T::Currency::withdraw(
                &appelant,
                T::FraisEnregistrement::get(),
                frame_support::traits::WithdrawReasons::FEE,
                frame_support::traits::ExistenceRequirement::KeepAlive,
            )?;

            let bloc_actuel = <frame_system::Pallet<T>>::block_number();
            Ressources::<T>::insert(
                &identifiant,
                (appelant.clone(), bloc_actuel),
            );

            Self::deposit_event(Event::RessourceEnregistrée {
                identifiant,
                propriétaire: appelant,
            });

            Ok(())
        }

        /// Transférer la propriété d'une ressource
        #[pallet::call_index(1)]
        #[pallet::weight(5_000)]
        pub fn transférer(
            origin: OriginFor<T>,
            identifiant: Vec<u8>,
            nouveau_propriétaire: T::AccountId,
        ) -> DispatchResult {
            let appelant = ensure_signed(origin)?;

            let (propriétaire_actuel, bloc) = Ressources::<T>::get(&identifiant)
                .ok_or(Error::<T>::RessourceIntrouvable)?;

            ensure!(
                propriétaire_actuel == appelant,
                Error::<T>::NonAutoriséÀTransférer
            );

            Ressources::<T>::insert(
                &identifiant,
                (nouveau_propriétaire.clone(), bloc),
            );

            Self::deposit_event(Event::RessourceTransférée {
                identifiant,
                ancien_propriétaire: appelant,
                nouveau_propriétaire,
            });

            Ok(())
        }
    }
}
```

## Smart Contracts ink! — Logique programmable sur la chaîne

ink! est un langage de smart contracts Rust-natif qui compile vers WASM et s'exécute sur Substrate.

```bash
# Créer un projet ink!
cargo contract new token-erc20
cd token-erc20
```

```rust
// lib.rs — Token ERC20 simplifié en ink!
#![cfg_attr(not(feature = "std"), no_std, no_main)]

#[ink::contract]
mod token_erc20 {
    use ink::storage::Mapping;

    #[ink(storage)]
    pub struct TokenERC20 {
        total_supply: Balance,
        balances: Mapping<AccountId, Balance>,
        allowances: Mapping<(AccountId, AccountId), Balance>,
        nom: ink::prelude::string::String,
        symbole: ink::prelude::string::String,
    }

    #[ink(event)]
    pub struct Transfer {
        #[ink(topic)]
        from: Option<AccountId>,
        #[ink(topic)]
        to: Option<AccountId>,
        value: Balance,
    }

    #[ink(event)]
    pub struct Approval {
        #[ink(topic)]
        owner: AccountId,
        #[ink(topic)]
        spender: AccountId,
        value: Balance,
    }

    #[derive(Debug, PartialEq, Eq, scale::Encode, scale::Decode)]
    #[cfg_attr(feature = "std", derive(scale_info::TypeInfo))]
    pub enum Erreur {
        SoldeInsuffisant,
        AllowanceInsuffisante,
    }

    pub type Résultat<T> = core::result::Result<T, Erreur>;

    impl TokenERC20 {
        /// Déploiement initial du contrat
        #[ink(constructor)]
        pub fn nouveau(
            supply_initiale: Balance,
            nom: ink::prelude::string::String,
            symbole: ink::prelude::string::String,
        ) -> Self {
            let appelant = Self::env().caller();
            let mut balances = Mapping::default();
            balances.insert(appelant, &supply_initiale);

            Self::env().emit_event(Transfer {
                from: None,
                to: Some(appelant),
                value: supply_initiale,
            });

            TokenERC20 {
                total_supply: supply_initiale,
                balances,
                allowances: Mapping::default(),
                nom,
                symbole,
            }
        }

        /// Balance d'un compte
        #[ink(message)]
        pub fn balance_de(&self, compte: AccountId) -> Balance {
            self.balances.get(compte).unwrap_or_default()
        }

        /// Transférer des tokens
        #[ink(message)]
        pub fn transférer(&mut self, destinataire: AccountId, montant: Balance) -> Résultat<()> {
            let expéditeur = self.env().caller();
            self.transférer_interne(expéditeur, destinataire, montant)
        }

        /// Approuver un spender
        #[ink(message)]
        pub fn approuver(&mut self, spender: AccountId, montant: Balance) -> Résultat<()> {
            let owner = self.env().caller();
            self.allowances.insert((owner, spender), &montant);

            self.env().emit_event(Approval {
                owner,
                spender,
                value: montant,
            });

            Ok(())
        }

        /// Transfert depuis un compte approuvé
        #[ink(message)]
        pub fn transfert_depuis(
            &mut self,
            from: AccountId,
            to: AccountId,
            montant: Balance,
        ) -> Résultat<()> {
            let appelant = self.env().caller();
            let allowance = self.allowances.get((from, appelant)).unwrap_or_default();

            if allowance < montant {
                return Err(Erreur::AllowanceInsuffisante);
            }

            self.allowances.insert((from, appelant), &(allowance - montant));
            self.transférer_interne(from, to, montant)
        }

        fn transférer_interne(
            &mut self,
            from: AccountId,
            to: AccountId,
            montant: Balance,
        ) -> Résultat<()> {
            let balance_from = self.balances.get(from).unwrap_or_default();

            if balance_from < montant {
                return Err(Erreur::SoldeInsuffisant);
            }

            self.balances.insert(from, &(balance_from - montant));
            let balance_to = self.balances.get(to).unwrap_or_default();
            self.balances.insert(to, &(balance_to + montant));

            self.env().emit_event(Transfer {
                from: Some(from),
                to: Some(to),
                value: montant,
            });

            Ok(())
        }
    }

    #[cfg(test)]
    mod tests {
        use super::*;

        #[ink::test]
        fn transfert_fonctionne() {
            let mut token = TokenERC20::nouveau(
                1000,
                "MonToken".into(),
                "MTK".into(),
            );
            let accounts = ink::env::test::default_accounts::<ink::env::DefaultEnvironment>();

            assert_eq!(token.balance_de(accounts.alice), 1000);
            assert!(token.transférer(accounts.bob, 100).is_ok());
            assert_eq!(token.balance_de(accounts.alice), 900);
            assert_eq!(token.balance_de(accounts.bob), 100);
        }

        #[ink::test]
        fn transfert_solde_insuffisant_échoue() {
            let mut token = TokenERC20::nouveau(100, "T".into(), "T".into());
            let accounts = ink::env::test::default_accounts::<ink::env::DefaultEnvironment>();

            assert_eq!(
                token.transférer(accounts.bob, 200),
                Err(Erreur::SoldeInsuffisant)
            );
        }
    }
}
```

## Intégration backend — Lire les événements on-chain

```toml
[dependencies]
subxt = "0.34"
```

```rust
// Observer les transferts de tokens en temps réel
use subxt::{OnlineClient, PolkadotConfig};

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let api = OnlineClient::<PolkadotConfig>::from_url("ws://localhost:9944").await?;

    // S'abonner aux nouveaux blocs
    let mut blocs = api.blocks().subscribe_finalized().await?;

    while let Some(bloc) = blocs.next().await {
        let bloc = bloc?;
        println!("Bloc #{} : {}", bloc.number(), bloc.hash());

        // Lire les événements du bloc
        let événements = bloc.events().await?;
        for événement in événements.iter() {
            let événement = événement?;
            println!("  Événement : {}.{}", événement.pallet_name(), événement.variant_name());
        }
    }

    Ok(())
}
```
