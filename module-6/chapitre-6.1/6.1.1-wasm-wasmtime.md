## WebAssembly n'est plus réservé aux navigateurs

WebAssembly (WASM) est un format binaire portable, compilé depuis Rust (ou d'autres langages), qui s'exécute dans un **sandbox** sécurisé. Côté serveur, il permet de :

- Charger des **plugins tiers** sans risque de compromettre le processus hôte
- Exécuter du **code utilisateur arbitraire** de manière sécurisée
- Déployer des **mises à jour de fonctionnalités** sans redémarrer le serveur
- Partager la **même logique métier** entre serveur et client (ex: règles de validation)

```
Architecture plugin WASM :
┌─────────────────────────────────────────────┐
│           Processus Hôte (Rust)             │
│  ┌──────────────┐   ┌──────────────────┐    │
│  │   API Axum   │   │   Wasmtime       │    │
│  │   :8080      │──►│   Runtime        │    │
│  └──────────────┘   │  ┌────────────┐  │    │
│                     │  │ Plugin A   │  │    │
│                     │  │ (sandbox)  │  │    │
│                     │  └────────────┘  │    │
│                     │  ┌────────────┐  │    │
│                     │  │ Plugin B   │  │    │
│                     │  │ (sandbox)  │  │    │
│                     │  └────────────┘  │    │
│                     └──────────────────┘    │
└─────────────────────────────────────────────┘

Les plugins WASM :
  ✅ Peuvent appeler les fonctions que l'hôte expose explicitement
  ❌ Ne peuvent pas accéder au système de fichiers de l'hôte
  ❌ Ne peuvent pas faire de syscalls directs
  ❌ Ne peuvent pas lire la mémoire de l'hôte
```

## Écrire un plugin WASM en Rust

```toml
# plugin-filtre/Cargo.toml
[package]
name = "plugin-filtre"
version = "0.1.0"
edition = "2021"

[lib]
crate-type = ["cdylib"]  # Dynamic library → .wasm

[dependencies]
# Pas de dépendances lourdes — le plugin doit être léger

[profile.release]
opt-level = "z"    # Optimisation taille
lto = true
strip = true
```

```rust
// plugin-filtre/src/lib.rs

/// Interface exposée par l'hôte (déclarée côté plugin)
extern "C" {
    fn hôte_log(ptr: i32, len: i32);
    fn hôte_obtenir_config(clé_ptr: i32, clé_len: i32, val_ptr: i32) -> i32;
}

/// Allouer de la mémoire (requis pour la communication hôte→plugin)
#[no_mangle]
pub extern "C" fn allouer(taille: usize) -> *mut u8 {
    let mut vec = Vec::with_capacity(taille);
    let ptr = vec.as_mut_ptr();
    std::mem::forget(vec); // Empêcher la désallocation
    ptr
}

/// Libérer de la mémoire
#[no_mangle]
pub extern "C" fn libérer(ptr: *mut u8, taille: usize) {
    unsafe { drop(Vec::from_raw_parts(ptr, taille, taille)); }
}

/// Fonction principale du plugin : filtrer une requête HTTP
/// Retourne 1 si la requête est autorisée, 0 sinon
#[no_mangle]
pub extern "C" fn filtrer_requête(données_ptr: i32, données_len: i32) -> i32 {
    let données = unsafe {
        std::slice::from_raw_parts(données_ptr as *const u8, données_len as usize)
    };

    // Désérialiser la requête
    let requête: RequêtePlugin = match serde_json_core::from_slice(données) {
        Ok((req, _)) => req,
        Err(_) => return 0,
    };

    // Logique de filtrage
    if requête.chemin.starts_with("/admin") && !requête.est_authentifié {
        log_plugin("Accès /admin bloqué : non authentifié");
        return 0;
    }

    if requête.ip.starts_with("10.0.0.") {
        log_plugin("IP interne — accès autorisé");
        return 1;
    }

    1 // Autoriser par défaut
}

fn log_plugin(message: &str) {
    unsafe {
        hôte_log(message.as_ptr() as i32, message.len() as i32);
    }
}

#[derive(serde::Deserialize)]
struct RequêtePlugin<'a> {
    chemin: &'a str,
    ip: &'a str,
    est_authentifié: bool,
}
```

## Charger et exécuter des plugins côté hôte

```toml
# Cargo.toml (hôte)
[dependencies]
wasmtime = "18"
wasmtime-wasi = "18"
```

```rust
// src/moteur_plugins.rs
use wasmtime::{Engine, Linker, Module, Store, Func, Memory};
use wasmtime_wasi::WasiCtxBuilder;
use std::collections::HashMap;
use std::path::Path;
use std::sync::Arc;
use tokio::sync::RwLock;

pub struct MoteurPlugins {
    engine: Engine,
    plugins: Arc<RwLock<HashMap<String, PluginChargé>>>,
}

struct PluginChargé {
    module: Module,
    nom: String,
    version: String,
}

impl MoteurPlugins {
    pub fn nouveau() -> anyhow::Result<Self> {
        let mut config = wasmtime::Config::new();
        config
            .async_support(true)
            .wasm_threads(false)   // Pas de threads dans les plugins
            .wasm_bulk_memory(true)
            .consume_fuel(true);  // Limite le temps d'exécution

        let engine = Engine::new(&config)?;

        Ok(MoteurPlugins {
            engine,
            plugins: Arc::new(RwLock::new(HashMap::new())),
        })
    }

    /// Charger un plugin depuis un fichier .wasm
    pub async fn charger(&self, nom: &str, chemin: &Path) -> anyhow::Result<()> {
        let octets = tokio::fs::read(chemin).await?;
        let module = Module::new(&self.engine, &octets)?;

        let plugin = PluginChargé {
            module,
            nom: nom.to_string(),
            version: "1.0.0".to_string(),
        };

        self.plugins.write().await.insert(nom.to_string(), plugin);
        tracing::info!(plugin = nom, chemin = %chemin.display(), "Plugin WASM chargé");
        Ok(())
    }

    /// Exécuter la fonction filtrer_requête d'un plugin
    pub async fn filtrer_requête(
        &self,
        nom_plugin: &str,
        données_requête: &[u8],
    ) -> anyhow::Result<bool> {
        let plugins = self.plugins.read().await;
        let plugin = plugins.get(nom_plugin)
            .ok_or_else(|| anyhow::anyhow!("Plugin '{}' introuvable", nom_plugin))?;

        let mut linker: Linker<ContextePlugin> = Linker::new(&self.engine);

        // Exposer les fonctions de l'hôte au plugin
        linker.func_wrap("env", "hôte_log",
            |mut caller: wasmtime::Caller<'_, ContextePlugin>, ptr: i32, len: i32| {
                let mémoire = caller.get_export("memory")
                    .and_then(|e| e.into_memory())
                    .expect("Mémoire WASM");

                let données = mémoire.data(&caller);
                let message = std::str::from_utf8(
                    &données[ptr as usize..(ptr + len) as usize]
                ).unwrap_or("message illisible");

                tracing::info!(plugin = "wasm", message = message, "Log plugin");
            }
        )?;

        let mut contexte = ContextePlugin {
            logs: Vec::new(),
        };

        let mut store = Store::new(&self.engine, contexte);
        store.add_fuel(1_000_000)?; // Limite à 1M instructions

        let instance = linker.instantiate_async(&mut store, &plugin.module).await?;

        // Copier les données dans la mémoire WASM
        let mémoire = instance.get_memory(&mut store, "memory")
            .ok_or_else(|| anyhow::anyhow!("Mémoire WASM introuvable"))?;

        // Allouer de la mémoire dans le plugin
        let fn_allouer = instance.get_typed_func::<u32, u32>(&mut store, "allouer")?;
        let ptr = fn_allouer.call_async(&mut store, données_requête.len() as u32).await?;

        // Écrire les données
        mémoire.write(&mut store, ptr as usize, données_requête)?;

        // Appeler la fonction de filtrage
        let fn_filtrer = instance.get_typed_func::<(i32, i32), i32>(
            &mut store, "filtrer_requête"
        )?;

        let résultat = fn_filtrer.call_async(
            &mut store,
            (ptr as i32, données_requête.len() as i32),
        ).await?;

        // Libérer la mémoire
        let fn_libérer = instance.get_typed_func::<(u32, u32), ()>(&mut store, "libérer")?;
        fn_libérer.call_async(&mut store, (ptr, données_requête.len() as u32)).await?;

        Ok(résultat == 1)
    }
}

struct ContextePlugin {
    logs: Vec<String>,
}
```

## Middleware Axum utilisant les plugins WASM

```rust
// src/middleware_wasm.rs
use axum::{extract::{Request, State}, middleware::Next, response::Response, http::StatusCode};
use std::sync::Arc;
use crate::moteur_plugins::MoteurPlugins;
use serde::Serialize;

#[derive(Serialize)]
struct DonnéesFiltrage<'a> {
    chemin: &'a str,
    méthode: &'a str,
    ip: &'a str,
    est_authentifié: bool,
}

pub async fn middleware_plugins_wasm(
    State(moteur): State<Arc<MoteurPlugins>>,
    req: Request,
    suivant: Next,
) -> Result<Response, StatusCode> {
    let chemin = req.uri().path().to_string();
    let méthode = req.method().as_str().to_string();

    // Extraire l'IP du client
    let ip = req.headers()
        .get("x-forwarded-for")
        .and_then(|v| v.to_str().ok())
        .unwrap_or("0.0.0.0")
        .to_string();

    let est_auth = req.extensions()
        .get::<crate::api::UtilisateurConnecté>()
        .is_some();

    let données = serde_json::to_vec(&DonnéesFiltrage {
        chemin: &chemin,
        méthode: &méthode,
        ip: &ip,
        est_authentifié: est_auth,
    }).unwrap();

    // Passer par le plugin de filtrage
    match moteur.filtrer_requête("filtre-sécurité", &données).await {
        Ok(true) => Ok(suivant.run(req).await),
        Ok(false) => {
            tracing::warn!(chemin = %chemin, ip = %ip, "Requête bloquée par plugin WASM");
            Err(StatusCode::FORBIDDEN)
        }
        Err(e) => {
            tracing::error!("Erreur plugin WASM : {}", e);
            // En cas d'erreur du plugin, fail-open (laisser passer)
            Ok(suivant.run(req).await)
        }
    }
}
```

## Rechargement à chaud des plugins

```rust
// Surveiller les modifications de fichiers et recharger
pub async fn surveiller_plugins(
    moteur: Arc<MoteurPlugins>,
    répertoire: std::path::PathBuf,
) {
    let mut intervalle = tokio::time::interval(std::time::Duration::from_secs(5));

    loop {
        intervalle.tick().await;

        let entrées = match tokio::fs::read_dir(&répertoire).await {
            Ok(e) => e,
            Err(_) => continue,
        };

        let mut flux = tokio_stream::wrappers::ReadDirStream::new(entrées);
        use tokio_stream::StreamExt;

        while let Some(Ok(entrée)) = flux.next().await {
            let chemin = entrée.path();
            if chemin.extension().map_or(false, |e| e == "wasm") {
                let nom = chemin.file_stem()
                    .and_then(|n| n.to_str())
                    .unwrap_or("inconnu");

                if let Err(e) = moteur.charger(nom, &chemin).await {
                    tracing::error!(plugin = nom, erreur = %e, "Échec rechargement plugin");
                } else {
                    tracing::info!(plugin = nom, "Plugin rechargé à chaud ✓");
                }
            }
        }
    }
}
```

## 🔥 Failure First — Plugin malveillant qui boucle

```rust
// plugin-malveillant/src/lib.rs
#[no_mangle]
pub extern "C" fn filtrer_requête(_: i32, _: i32) -> i32 {
    // Boucle infinie — tente de bloquer l'hôte
    loop {
        let _ = 1 + 1;
    }
}
```

Avec `store.add_fuel(1_000_000)`, Wasmtime interrompt l'exécution après 1M instructions :
```
Error: "fuel" ran out: out of fuel executing wasm
```

L'hôte continue de fonctionner normalement. Le plugin malveillant est neutralisé.

## Résumé

| Capacité WASM | Implémentation |
|---|---|
| Sandbox sécurisée | Wasmtime par défaut |
| Limite CPU | `store.add_fuel()` |
| Limite mémoire | `store.limiter()` |
| Isolation réseau | WASI ne donne accès réseau que si explicitement accordé |
| Rechargement à chaud | Recharger le `Module` sans redémarrer |
| Communication hôte↔plugin | Mémoire partagée + fonctions importées/exportées |
